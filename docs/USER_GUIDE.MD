# User Guide

## Quick Start

### View Your Portfolio
The dashboard shows three key metrics:
- **Total Value** - Current worth of all positions
- **Total Gain/Loss** - Overall profit/loss with percentage
- **Total Cost** - Your total investment

### Add a Stock

1. Click **"Add Stock"** button
2. Search by symbol (e.g., "AAPL") or company name
3. Click **"Select"** on the stock you want
4. Enter:
   - Number of shares
   - Purchase price (defaults to current price)
5. Click **"Add to Portfolio"**

### Refresh Prices

- **Refresh All** - Updates all stock prices
- **Per-stock refresh** - Click the refresh icon in any row

### Understanding the Holdings Table

Each row shows:
- Stock symbol and company
- Shares owned
- Your average purchase price
- Current market price
- Total value and gain/loss
